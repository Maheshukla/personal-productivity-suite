# Personal Productivity Suite

## Project Overview

This is a Python-based Personal Productivity Suite designed to help users manage daily tasks efficiently. It includes multiple tools integrated into a single command-line interface.

---

## Features

* 📝 Notes Manager (Add, View, Delete Notes)
* 💾 Automatic Data Saving (JSON)
* 🔄 Backup System for Notes
* 🧮 Calculator (Basic Operations)
* ⏱️ Timer / Countdown Tool
* 🔢 Unit Converter (KM → Meter, Celsius → Fahrenheit)
* 📁 File Organizer (Organizes files by type)

---

## Technologies Used

* Python (Core Programming)
* JSON (Data Storage)
* OS & Shutil Modules (File Handling)
